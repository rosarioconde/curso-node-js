# Pre-Entrega Node.js 
## Gestión de Productos con FakeStore API

Este proyecto es una aplicación de consola desarrollada con Node.js que permite consultar, crear y eliminar productos utilizando la API pública de FakeStore.

El programa interpreta comandos ingresados desde la terminal mediante `process.argv` y realiza peticiones HTTP asíncronas con `fetch`.

---

## Objetivos del Proyecto

- Configurar un proyecto Node.js con ES Modules.
- Utilizar `process.argv` para capturar argumentos de la terminal.
- Consumir una API REST utilizando `fetch`.
- Implementar operaciones:
  - GET
  - POST
  - DELETE
- Manejar errores con `try/catch`.

---

## 🛠️ Tecnologías Utilizadas

- Node.js
- JavaScript (ES Modules)
- FakeStore API
